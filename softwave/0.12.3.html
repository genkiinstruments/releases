Softwave - v0.12.3

A new version of Softwave is available!

This version contains a firmware update solving some stability issues when working with a DAW, as well as fixing a bug introduced in version 0.12.2, where Wave could not be connected on Windows 10 systems.

Main highlights:

  • Fixed a bug where Wave could not be connected on Windows 10 systems.
  • Wave would frequently disconnect when working within a DAW that might send MIDI SysEx messages to Wave.
  • Virtual MIDI driver version is displayed in the “About” window on Windows.
  • Fixed a broken link in the “Help” menu item
  • Fixed a bug where undoing a delete of a function in a preset would display it in the wrong slot.
