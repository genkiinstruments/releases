Softwave - v0.13.6

A new version of Softwave is available!

This update includes a firmware update for Wave.

Changes in Wave firmware v1.7.11

  • Fixed a bug where MIDI Program change messages were incorrectly sent in Standalone Mode
  • Fixed a bug where auto-activation of presets was not treated in Standalone Mode
  • Fixed a bug affecting Momentery/Momentoggle button actions

Changes in Softwave v0.13.6:

  • Added “Reset Bluetooth Cache” option to the menu for macOS
  • Added “Reset Softwave Settings” option to the menu
